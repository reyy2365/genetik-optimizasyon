# Genetik Algoritma ile Optimizasyon

Bu projede tarım arazisinde gübre ve sulama miktarlarının,
bitki verimini maksimize edecek şekilde genetik algoritma
kullanılarak optimizasyonu gerçekleştirilmiştir.

## Senaryo
Tarım Arazisinde Gübre ve Sulama Ayarı (Senaryo 4)

## Amaç
Gerçek hayattaki kısıtlar altında maksimum verimi sağlayan
optimum çözümü bulmak.

## Kullanılan Yöntem
- Genetik Algoritma
- Amaç Fonksiyonu ile Maksimizasyon
- Kısıtlı Optimizasyon

## Kullanılan Teknolojiler
- Python
- Jupyter Notebook
- NumPy
